#项目
1. 生成项目
```
npm install express-generator -g
express -e blog
```
2. 安装依赖 
```
cd blog && npm install
```
3. 启动项目
```
SET DEBUG=blog:* & npm start
```


#安装bootstrap
```
touch .bowerrc            创建配置文件指定下载后的包的存放目录
bower init                初始化bower.json配置文件
bower install bootstrap --save   安装bootstrap
```
